# SAYA.GGH - Portofolio Arsitektur & Desain

Ini adalah website portofolio arsitektur profesional yang dibangun menggunakan **Next.js**, **Tailwind CSS**, dan **Supabase**. Website ini dilengkapi dengan galeri proyek dinamis, sistem manajemen konten (CMS) untuk admin, dan integrasi dengan Google Gemini AI untuk pembuatan konten.


## ğŸ—„ï¸ Persiapan Database (Supabase)

Jalankan kode SQL berikut di **SQL Editor** Supabase Anda untuk membuat tabel-tabel yang diperlukan:

### Tabel-tabel
```sql
-- Tabel Projects
create table projects (
  id bigint generated by default as identity primary key,
  title text not null,
  location text,
  year text,
  status text default 'On Going',
  description text,
  tags text[],
  image_url text,
  created_at timestamp with time zone default now()
);

-- Konten Halaman (CMS)
create table page_content (
  section text primary key,
  title text,
  subtitle text,
  body text,
  image_url text,
  updated_at timestamp with time zone default now()
);

-- Tabel Tim (Team)
create table team (
  id bigint generated by default as identity primary key,
  name text not null,
  role text,
  image_url text,
  created_at timestamp with time zone default now()
);

-- Tabel Berita (News)
create table news (
  id bigint generated by default as identity primary key,
  title text not null,
  date date default now(),
  content text,
  image_url text,
  created_at timestamp with time zone default now()
);
```

### Penyimpanan (Storage)
1. Buat **Public Bucket** bernama `images`.
2. Atur **RLS Policies** pada bucket tersebut untuk mengizinkan `INSERT`, `SELECT`, `UPDATE`, dan `DELETE` bagi peran `anon` dan `authenticated` (untuk keperluan pengembangan).

## ğŸ“ Struktur Proyek

### `FrontEnd/app`

- `supabase.ts`: Konfigurasi klien Supabase.
- `gemini.ts`: Fungsi pembantu Google Gemini AI.
