# SAYA.GGH - Portofolio Arsitektur & Desain

Website portofolio arsitektur profesional berbasis **Next.js**, **Tailwind CSS**, dan **Supabase**. Fitur utama mencakup galeri proyek dinamis, CMS admin, dan integrasi Google Gemini AI untuk pembuatan konten.

## Teknologi Utama

- **Frontend:** Next.js (App Router), TypeScript, Tailwind CSS, Flowbite React.
- **Backend/Database:** Supabase (PostgreSQL, Storage, Auth).
- **Integrasi AI:** Google Gemini AI (untuk pembuatan deskripsi otomatis).
- **Ikon:** React Icons.

---

## Instalasi & Persiapan

### 1. Prasyarat

Pastikan Anda telah menginstal hal berikut:

- [Node.js](https://nodejs.org/) (v18 atau lebih baru)
- [Git](https://git-scm.com/)

### 2. Clone dan Instal Dependensi

```bash
git clone https://github.com/SpaceMan586/SAYA.GGH
cd P3L

# Instal dependensi FrontEnd
cd FrontEnd
npm install

# (Opsional) Instal dependensi BackEnd jika menggunakan server kustom
cd ../BackEnd
npm install
```

### 3. Variabel Lingkungan (Environment Variables)

Buat file `.env.local` di dalam direktori `FrontEnd`:

```env
NEXT_PUBLIC_SUPABASE_URL=url_supabase_anda
NEXT_PUBLIC_SUPABASE_ANON_KEY=key_anon_supabase_anda
NEXT_PUBLIC_GEMINI_API_KEY=api_key_gemini_anda
```

## Persiapan Database (Supabase)

Jalankan kode SQL berikut di **SQL Editor** Supabase Anda untuk membuat tabel-tabel yang diperlukan:

### Tabel-tabel

```sql
-- Tabel Projects
create table projects (
  id bigint generated by default as identity primary key,
  title text not null,
  location text,
  year text,
  status text default 'On Going',
  description text,
  tags text[],
  image_url text,
  created_at timestamp with time zone default now()
);

-- Konten Halaman (CMS)
create table page_content (
  section text primary key,
  title text,
  subtitle text,
  body text,
  image_url text,
  updated_at timestamp with time zone default now()
);

-- Tabel Tim (Team)
create table team (
  id bigint generated by default as identity primary key,
  name text not null,
  role text,
  image_url text,
  created_at timestamp with time zone default now()
);

-- Tabel Berita (News)
create table news (
  id bigint generated by default as identity primary key,
  title text not null,
  date date default now(),
  content text,
  image_url text,
  created_at timestamp with time zone default now()
);
```

### Penyimpanan (Storage)

1. Buat **Public Bucket** bernama `images`.
2. Atur **RLS Policies** pada bucket tersebut untuk mengizinkan `INSERT`, `SELECT`, `UPDATE`, dan `DELETE` bagi peran `anon` dan `authenticated` (untuk keperluan pengembangan).

## Menjalankan Aplikasi

### Menjalankan Server Pengembangan

```bash
cd FrontEnd
npm run dev
```

Buka `http://localhost:3000` di browser Anda.

### Dashboard Admin

Akses CMS melalui alamat `/admin/login`.

- **Kredensial Default (Demo):** `admin@saya.ggh` / `admin`

## Daftar Route & Link Halaman

Semua link di bawah memakai base URL lokal: `http://localhost:3000`

| Halaman | Route | Link |
| --- | --- | --- |
| Home | `/` | [http://localhost:3000/](http://localhost:3000/) |
| About | `/about` | [http://localhost:3000/about](http://localhost:3000/about) |
| Project List | `/project` | [http://localhost:3000/project](http://localhost:3000/project) |
| Project Detail (dynamic) | `/project/[id]` | Contoh: [http://localhost:3000/project/1](http://localhost:3000/project/1) |
| News List | `/news` | [http://localhost:3000/news](http://localhost:3000/news) |
| News Detail (dynamic) | `/news/[id]` | Contoh: [http://localhost:3000/news/1](http://localhost:3000/news/1) |
| Admin Root (redirect) | `/admin` | [http://localhost:3000/admin](http://localhost:3000/admin) -> redirect ke `/admin/login` |
| Admin Login | `/admin/login` | [http://localhost:3000/admin/login](http://localhost:3000/admin/login) |
| Admin Dashboard | `/admin/dashboard` | [http://localhost:3000/admin/dashboard](http://localhost:3000/admin/dashboard) |
| Admin Debug | `/admin/debug` | [http://localhost:3000/admin/debug](http://localhost:3000/admin/debug) |

Catatan:
- Route group Next.js seperti `/(public)` dan `/(main)` tidak muncul di URL akhir.
- Untuk route dynamic (`[id]`), ganti `id` dengan data valid dari database (mis. `1`, `2`, dst).

## Struktur Proyek

### `FrontEnd/app`

- `/(public)`: Layout dan halaman publik.
- `/(public)/(main)`: Halaman utama seperti `about`, `project`, `news`.
- `/(public)/(sidebar)`: Layout yang memakai sidebar.
- `/admin`: Login dan Dashboard CMS.
- `page.tsx`: Halaman publik utama.

### `FrontEnd/lib`

- `supabase.ts`: Konfigurasi klien Supabase.
- `gemini.ts`: Fungsi pembantu Google Gemini AI.

## Fitur Utama

- **Background Dinamis**: Ubah gambar dan teks halaman utama langsung dari dashboard.
